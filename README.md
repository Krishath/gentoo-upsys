# gentoo-upsys
Gentoo System Upgrade Script
	
Before using gentoo-upsys you need to:
```
emerge eix -av
```
Usage:
```
	gentoo-upsys [options]
	---
	Options:
	-n | excludes sync of the gentoo tree
	-e [ebuild] | excludes the specified ebuild
	-q | compiles quietly
	-v | verbose listings (can be used with -q)
```
